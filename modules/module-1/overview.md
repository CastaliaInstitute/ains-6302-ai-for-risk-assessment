# Module 1: Quantitative Risk and Vulnerability Prioritization

## Overview

Translates qualitative risk talk into numbers executives can act on. Students build FAIR-style loss distributions, apply EPSS and KEV feeds to prioritize patching, and critique the assumptions behind each score.

## Topics

- The FAIR model: loss event frequency and magnitude
- CVSS, EPSS, and the KEV catalog — what each actually measures
- Monte Carlo simulation of annualized loss
- Prioritization as a budgeted optimization problem

## Module Deliverable

Given a synthetic asset inventory and a CVE feed, build a FAIR-style Monte Carlo simulator and produce a patch prioritization ranked by expected loss reduction per dollar.
