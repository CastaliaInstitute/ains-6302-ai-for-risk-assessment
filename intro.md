# AINS6302 AI for Risk Assessment

This book is the instructional hub for `AINS6302 AI for Risk Assessment` in the
**Cybersecurity AI** specialization of the Aurnova MSAI program.

It is organized as one unified MyST Jupyter Book so learners and
instructors navigate the same course artifact while accessing
different kinds of material:

- book prose for conceptual understanding
- assignments for applied practice
- slides for presentation flow
- narration for asynchronous delivery
- instructor notes for teaching guidance
- notebooks for executable exploration

Teaches AI-based risk modeling and predictive analysis for cybersecurity decision-making. Covers quantitative risk (FAIR), vulnerability prioritization, attack-path analysis on graphs, and forecasting breach likelihood under uncertainty.

The first template version focuses on two foundational modules:

1. Quantitative Risk and Vulnerability Prioritization
2. Attack-Path Modeling and Breach Forecasting

## Where this runs

- **Book site** — GitHub Pages, published from `main` on every push
- **Autograder** — [Anubis LMS](https://anubis.castalia.institute)
  at `ains6302`. Students receive a private repository generated
  from this template, push code, and get autograded feedback.
